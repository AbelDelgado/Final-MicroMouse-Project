\subsection{Detailed Description}
Motors interface. 

\begin{DoxyAuthor}{Author}
Madzivire Chikwanda 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
22 April 2015
\end{DoxyDate}
Methods for controling motors. The T\+I\+M\+E\+R module has to be initialized by calling \hyperlink{init_8h_a2858154e2009b0e6e616f313177762bc}{init()} , defined in \hyperlink{init_8h}{init.\+h}, before methods defined in this file can be called.

The left motor is interfaced with P\+O\+R\+T B, P\+W\+M channel 4, \& left wheel encoder is interfaced to Pulse Accummulator A (P\+A\+C\+N32). The right motor is interfaced with P\+O\+R\+T A, P\+W\+M channel 5, \& left wheel encoder is interfaced to Pulse Accummulator B (P\+A\+C\+N10).
