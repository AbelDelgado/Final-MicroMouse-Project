\subsection{Detailed Description}
\hyperlink{struct_maze}{Maze} application -\/ uses Flood Fill Algorithm to solve a maze. 

\begin{DoxyAuthor}{Author}
Madzivire Chikwanda  No known bugs. 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
21 March 2015
\end{DoxyDate}
This a C adaptation for the H\+C\+S12 microcontroler of Maze.\+class posted on the Udemy website for the online class\textquotesingle{}Artificial Intelligence with Arduino.\textquotesingle{}

The application simulates a maze which is mapped and solve by a \textquotesingle{}mouse.\textquotesingle{} Each square on the maze is refered to as cell, walls along the y-\/axis are considered horizontal, and walls along the x-\/axis are vertical. \hyperlink{struct_maze}{Maze} size is is defined by the R\+O\+W\+S \& C\+O\+L\+U\+M\+N\+S macros.

The maze is solved by using a flood-\/fill algorithm to map and find a path to the Target \hyperlink{struct_cell}{Cell}.
