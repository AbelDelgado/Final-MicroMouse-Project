Detailed Description

Entry point for the HCS12 maze application.

Author
Madzivire Chikwanda This implementation was not completely tested.
Date
22 April 2015

Calls to methods in main_asm.h, init.h, ADsensors.h, PWMmotors.h, & maze.h to solve a maze are made here.

  • a call to init() has to be made before any call to scanWalls(), turnToBestNeighbor(), or any other methods in PWMmotors.h, or ADsensors.h
  • the Maze struct has to be initialiazed by calling the initMaze() method before any other calls to methods in maze.h
