Detailed Description

Maze application - uses Flood Fill Algorithm to solve a maze.

Author
Madzivire Chikwanda No known bugs.
Date
21 March 2015

This a C adaptation for the HCS12 microcontroler of Maze.class posted on the Udemy website for the online class'Artificial Intelligence with Arduino.'

The application simulates a maze which is mapped and solve by a 'mouse.' Each square on the maze is refered to as cell, walls along the y-axis are considered horizontal, and walls along the x-axis are vertical. Maze size is is defined by the ROWS & COLUMNS macros.

The maze is solved by using a flood-fill algorithm to map and find a path to the Target Cell.

See also
https://www.udemy.com/nanomouse/
