Author
Madzivire Chikwanda

The following modules and registers are initialized by the init routine

  • ADT1 Analog to Digital module
  • PORT A & B
  • TIMER module
    • Pulse width modulation (PWM) registers
    • Pulse Accumulator (PAC) registers
